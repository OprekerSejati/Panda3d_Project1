# AI Swarm Simulation – Panda3D

A modular 3D **AI swarm simulation** built with [Panda3D](https://www.panda3d.org/), featuring hundreds of agents, obstacles, and interactive controls.

## Features

- ✅ **Desert environment** with full proporsional 2048×2048 background
- ✅ **Cube player** with keyboard movement (WASD)
- ✅ **200 autonomous agents** demonstrating basic steering behaviors
- ✅ **Obstacles** randomly generated in the scene
- ✅ **Stats UI** showing agent count, collisions, FPS, and simulation time
- ✅ **Modular project structure** ready for extension
